CSS Grid Layout

CSS Grid Layout is a powerful two-dimensional layout system that provides a grid-based structure for designing web pages. It allows designers and developers to create complex layouts more easily than traditional methods, offering precise control over both rows and columns.

Key Concepts

  • Grid Container: The element that establishes a grid formatting context for its child elements. It is defined by setting the display property to grid or inline-grid.
  • Grid Items: Child elements of the grid container. They are automatically placed into the grid layout based on the container's properties.
  • Grid Lines: Lines that form the boundaries between columns and rows. They are used to position grid items. Grid lines are indexed starting from 1 and can be referenced in CSS.
  • Grid Tracks: The space between two grid lines, which is either a column or a row. Tracks are created automatically based on the size of the grid container.
  • Grid Cells: The intersection of a row and a column, forming a single unit of the grid where a grid item can be placed.
  • Grid Areas: Rectangular areas within the grid, defined by combining multiple grid cells. Grid items can span across multiple grid areas.

CSS Grid Properties

Container Properties

  • grid-template-columns and grid-template-rows: Define the size of columns and rows in the grid.
  • grid-template-areas: Defines named grid areas within the grid layout.
  • grid-auto-rows and grid-auto-columns: Specify the size of rows and columns that are created implicitly.
  • gap: Sets the space between rows and columns.

Item Properties

  • grid-column-start and grid-column-end: Define where a grid item starts and ends along the column lines.
  • grid-row-start and grid-row-end: Define where a grid item starts and ends along the row lines.
  • grid-area: A shorthand property for setting grid-row-start, grid-column-start, grid-row-end, and grid-column-end.

Functions

  • repeat(): Simplifies the definition of repeated tracks.
  • minmax(): Sets a size range for grid tracks.
  • fit-content(): Sizes the grid item based on its content, but within a defined maximum size.
