CSS Grid Container

The CSS Grid Container is an element that establishes a grid formatting context for its children, allowing for a two-dimensional layout system.

Key Properties

  • display: Defines the grid container. Use display: grid; or display: inline-grid; to create a grid container.
  • grid-template-columns: Specifies the number and size of columns in the grid.
  • grid-template-rows: Specifies the number and size of rows in the grid.
  • grid-template-areas: Defines named grid areas that can be used to place grid items.
  • grid-gap: Sets the space between rows and columns in the grid.
  • justify-items: Aligns grid items along the row axis.
  • align-items: Aligns grid items along the column axis.
